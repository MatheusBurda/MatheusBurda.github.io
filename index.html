BallCollision

Simulação 3D de colisão elástica entre bolas dentro de um cubo. Este projeto foi meu primeiro contato com Python e PyOpenGL. Após alguns dias com 30 abas abertas no navegador lendo a documentação do OpenGL e aprendendo Python, o simulador saiu do papel. Foi o primeiro projeto que postei publicamente no Github. E produzi um vídeo para o YouTube explicando um pouco mais sobre o código.

Tecnologias utilizadas:

  • Python
  • OpenGL
  • Github

Carro Controlado por Smartphone

Na matéria Microcontroladores 1, no curso de Técnico em Eletrônica, tive que desenvolver um projeto. A minha ideia foi fazer um carro de controle remoto controlado pelo celular. Fui desafiado pelo professor a fazer minha própria interface para controlar o carrinho. Então, usando o MIT App Inventor, eu fiz meu próprio App. Usei o Arduíno Uno, que conectava os motores do carrinho com o celular via Bluetooth. Um dos links abaixo leva para o YouTube, onde eu explico melhor esse projeto e mostro ele funcionando.

Tecnologias utilizadas:

  • Arduíno Uno
  • MIT App Inventor

Pong

Pong foi o primeiro jogo que programei e é inspirado no famoso jogo de Atari. Feito com a linguagem C, ele é jogado e 100% impresso no terminal, sem usar bibliotecas gráficas. O código é amador, inclusive possui trechos que pecam em semântica, mas como foi o primeiro programa razoavelmente mais complexo que fiz eu achei importante deixar nessa lista.

Tecnologias utilizadas:

  • Linguagem C
  • Impressão em qualquer posição do terminal
  • Impressão colorida no terminal
