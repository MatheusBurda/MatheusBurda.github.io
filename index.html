Site próprio

Este site é 100% minha autoria e feito com conceito Mobile First. Sempre tive curiosidade de como seria fazer um site em HTML e CSS, juntei o útil ao agradável e fiz meu próprio site. Já havia construído outros 3 sites, porém usando construtores de site de empresas de hospedagem. Campo Minado

Me baseei no jogo de campo minado que jogava no Windows antigamente para criar meu próprio. Após ter feito o BallCollision eu ainda estava com vontade de aprender coisas novas na quarentena. Um professor havia comentado sobre a biblioteca gráfica SDL, então a usei para fazer este jogo. As classes do c++ estão acopladas e não seguem muito bem o padrão orientado a objetos, mas o resultado final está bem agradável.

Tecnologias utilizadas:

  • C++
  • SDL 2

BallCollision

Simulação 3D de colisão elástica entre bolas dentro de um cubo. Este projeto foi meu primeiro contato com Python e PyOpenGL. Após alguns dias com 30 abas abertas no navegador lendo a documentação do OpenGL e aprendendo Python, o simulador saiu do papel. Foi o primeiro projeto que postei publicamente no Github. E produzi um vídeo para o YouTube explicando um pouco mais sobre o código.

Pong

Pong é inspirado no famoso jogo de Atari e foi o primeiro jogo que programei. Feito em linguagem C, ele é jogado e 100% impresso no terminal, sem usar bibliotecas gráficas. O código é amador, inclusive possui trechos que pecam em semântica, mas como foi o primeiro programa razoavelmente mais complexo que fiz eu achei importante deixar nessa lista.

Tecnologias utilizadas:

  • Linguagem C
  • Impressão em qualquer posição do terminal
  • Impressão colorida no terminal

Carro Controlado por Smartphone

Na matéria Microcontroladores 1, no curso de Técnico em Eletrônica, tive que desenvolver um projeto. A minha ideia foi fazer um carro de controle remoto controlado pelo celular. Fui desafiado pelo professor a fazer minha própria interface para controlar o carrinho. Então, usando o MIT App Inventor, eu fiz meu próprio App. Usei o Arduíno Uno, que controlava os motores do carrinho e se comunicava via Bluetooth com o celular. No vídeo do YouTube eu explico melhor o funcionamento e mostro mais detalhes.

Tecnologias utilizadas:

  • Arduíno Uno
  • MIT App Inventor
